# DnD-DragonDenPPT

A modern, interactive presentation for Dragon Den pitches in Dungeons & Dragons campaigns.

## Project Overview

This project features a sleek, single-page web application designed as a presentation tool for pitching products or ideas within a D&D universe. The presentation follows a fantasy-themed narrative centered around "HealthUp" - a revolutionary healing product for adventurers.

## Features

- **Interactive Slides**: Navigate through 10 professionally designed slides using arrow keys or navigation buttons
- **Chapter Navigation**: Sidebar with organized sections for easy access to specific topics
- **Responsive Design**: Adapts to different screen sizes with scalable typography
- **Modern UI**: Dark-themed interface with golden accents, inspired by fantasy aesthetics
- **Keyboard Controls**: Use arrow keys for smooth slide navigation

## Presentation Structure

The presentation walks through a complete product pitch narrative:

1. **The Problem**: Illustrates issues with traditional healing methods in D&D
2. **The Solution**: Introduces HealthUp as a revolutionary alternative
3. **Product Details**: Explains the HealthUp system and benefits
4. **Differentiation**: Compares HealthUp with traditional vials
5. **Business Model**: Shows the refill/repeat customer system
6. **Investment Ask**: Presents expansion opportunities

## Technical Details

- Pure HTML/CSS/JavaScript implementation (no external dependencies)
- Self-contained in a single HTML file for easy deployment
- Custom SVG noise texture for visual depth
- CSS animations for smooth transitions
- Mobile-friendly touch swipe support

## How to Use

1. Open `index.html` in any modern web browser
2. Navigate slides using:
   - Right/Down arrow keys or spacebar for next slide
   - Left/Up arrow keys for previous slide
   - Sidebar chapter navigation
   - On-screen navigation buttons

## Development

To modify the presentation:

1. Edit the HTML structure within the slide divs
2. Adjust styling in the CSS section
3. Modify slide navigation or behavior in the JavaScript section

All code is contained within `index.html` for simplicity.

## Deployment

Simply host the `index.html` file on any web server or open it directly in a browser. No build process required.
